# Stadium Jukebox Website

This is a ready-to-upload Vite + React website.

## Easiest way to launch

### Option A: Vercel (easiest)
1. Create a GitHub repo
2. Upload all files from this zip
3. Go to Vercel
4. Import the GitHub repo
5. Click Deploy

### Option B: Netlify
1. Create a GitHub repo
2. Upload all files from this zip
3. Go to Netlify
4. Add new project from GitHub
5. Build command: `npm run build`
6. Publish directory: `dist`

## Run locally
```bash
npm install
npm run dev
```
